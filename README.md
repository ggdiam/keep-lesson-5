## Как запустить

Клонировать репо:

```
git clone git@github.com:ggdiam/keep-lesson-5.git
```

Перейти в папку keep-lesson-5
```
cd keep-lesson-5
```

Установить библиотеки из npm
```
npm install
```

Запустить
```
npm start
```

В браузере откроется приложение по адресу:<br>
[http://localhost:3000](http://localhost:3000)
